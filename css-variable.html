独学でエンジニアになるサイト

CSSの変数を学んだのでボタン一つでテーマの色を変えてるページを作成したぞ。

dotinstallでCSSの変数を学ぶ

CSSの変数を学んだことでボタン一つでテーマの色合いを変化させることができるようになりました。具体的にはbodyの背景色とh1,h2,p,footerの色がボタンを押下する度に切り替わります。(変化を見るためにメニューの文字はあえてそのままにしました。）
↓のボタンを押下する度に、色調の整ったテーマカラーがランダムに切り替わります。

Button

Button だけ強調されているのは、色相が反転するようにランダムに出てくる数値からcalcを使って−１８０させているからです。−１８０だと、どぎつく感じるようであれば、１８０を９０とかに変えるのもありとのことです。

Webサイトの色相を合わせる方法は今までhue360を使う方法しか知らなかったのですが、この方法を使えば色相に対する知識が無くても色相の合うサイトを作ることができそうです。

JavaScriptの理解はまだまだ

getElementById、addEventListener、Math.random()とかはそれぞれ個別で何をしているか理解しているけれど、ボタンを押す度に１〜３６０の数値を--my-hueに代入しろという命令を空で書くことはまだまだできそうにない。
